29 /** \c h_item<int dimensions> identifies an instance of a \c
30  group::parallel_for_work_item function object executing at each
31  point in a local \c range<int dimensions> passed to a \c
32  parallel_for_work_item call or to the corresponding \c
33  parallel_for_work_group call if no range is passed to the \c
34  parallel_for_work_item call. It encapsulates enough information to
35  identify the work-item’s local and global items according to the
36  information given to \c parallel_for_work_group (physical ids) as
37  well as the work-item’s logical local items in the flexible
38  range. All returned items objects are offset-less. Instances of
39  the \c h_item<int dimensions> class are not user-constructible and
40  are passed by the runtime to each instance of the function object.
41 */
