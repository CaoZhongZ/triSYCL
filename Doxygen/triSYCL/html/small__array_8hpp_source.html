41 /** Define a multi-dimensional index, used for example to locate a work
42  item or a buffer element
43 
44  Unfortunately, even if std::array is an aggregate class allowing
45  native list initialization, it is no longer an aggregate if we derive
46  from an aggregate. Thus we have to redeclare the constructors.
47 
48  \param BasicType is the type element, such as int
49 
50  \param Dims is the dimension number, typically between 1 and 3
51 
52  \param FinalType is the final type, such as range<> or id<>, so that
53  boost::operator can return the right type
54 
55  \param EnableArgsConstructor adds a constructors from Dims variadic
56  elements when true. It is false by default.
