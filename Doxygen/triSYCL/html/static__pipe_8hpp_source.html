28 /** A SYCL static-scoped pipe equivalent to an OpenCL program-scoped
29  pipe
30 
31  Implement a FIFO-style object that can be used through accessors
32  to send some objects T from the input to the output.
33 
34  Compared to a normal pipe, a static_pipe takes a constexpr size
35  and is expected to be declared in a compile-unit static context so
36  the compiler can generate everything at compile time.
37 
38  This is useful to generate a fixed and optimized hardware
39  implementation on FPGA for example, where the interconnection
40  graph can be also inferred at compile time.
41 
42  It is not directly mapped to the OpenCL program-scoped pipe
43  because in SYCL there is not this concept of separated
44  program. But the SYCL device compiler is expected to generate some
45  OpenCL program(s) with program-scoped pipes when a SYCL
46  static-scoped pipe is used. These details are implementation
47  defined.
