70  /** Flattening helper that does not change scalar values but flatten a
71  vec<T, n> v into a tuple<T, T,..., T>{ v[0], v[1],..., v[n-1] }
72 
73  If we have a vector, just forward its array content since an array
74  has also a tuple interface :-) (23.3.2.9 Tuple interface to class
75  template array [array.tuple])

78  static auto flatten(const vec<Element, s> i) {
79  static_assert(s <= V::dimension,
