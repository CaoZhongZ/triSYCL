Detailed Description

template<std::size_t PDim = 1>
struct cl::sycl::xilinx::partition::complete< PDim >

Represent a complete partition.

The single array would be partitioned into individual elements. That is if we have a 4-element array with one dimension, the array is completely partitioned into distributed RAM or 4 independent registers.

Parameters
PDimis the dimension that user wants to apply complete partition on. If PDim is 0, all dimensions will be completely partitioned.
