395  caller of this function, the \c buffer_waiter, so there is no
396  need to get a \ future otherwise there will be a dead-lock if

404  if ((shared_from_this().use_count() > 2) &&
405  modified && (final_write_back || data_host)) {
406  // Create a promise to wait for
